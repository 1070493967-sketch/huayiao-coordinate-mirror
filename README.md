# 文昌星 — 认知诊断工具（话疗罗盘）

你不是需要被"诊断"——你只是需要一个地方，把心里的话说出来，然后看看自己到底在哪儿。

文昌星主智慧、点化——帮人看清楚自己在哪里。话疗罗盘就是它的工具化版本。

话疗罗盘是一个**对话式自我定位工具**。它不是一个测试题，不是一个量表——它是一次对话。你跟AI聊10分钟左右，它会帮你理清：

- 你现在卡在什么地方
- 你的惯性盲区是什么
- 你可以往哪个方向走
- 第一步可以做什么（很小的那种）

**v2.2 新增**：对话结束后，AI会给你一段可以直接复制发朋友圈的坐标快照文案。想要专属坐标图的，加创作者微信，他会直接发给你。

---

## 它跟别的"性格测试"有什么不同？

市面上大多数性格测试（MBTI、九型人格、大五人格）告诉你"你是什么"——然后你就被那个标签框住了。

话疗罗盘不贴标签。它认为**人不是固定的——人是一条河，不是一棵树。**

它给你的是**当前位置和可能的流向**，不是"你是一个什么样的人"。

## 快速开始

### 跟AI对话（推荐）

1. 将此 skill 安装到你的AI助手（WorkBuddy 等支持 skill 安装的平台）
2. 直接对 AI 说一句：**"我不太清楚自己现在怎么了"**
3. AI 会引导你完成一次约 10 分钟的对话
4. 对话结束后，你会获得：
   - 你的当前位置、盲区、方向和第一步
   - **一段可直接复制发朋友圈的坐标快照文案**
   - **引导你加创作者微信，领取专属坐标图**

### 手动自诊

如果你想自己研究，`SKILL.md` 文件里包含完整的对话流程和判定逻辑。你也可以阅读 `scripts/36格参考数据.md` 来查看 36 个常见坐标的详细描述。

## 原理

这套工具基于一个原创的认知框架——**四性六层体系**，包含：

- **四性双重维度**：每个人在兽性、人性、神性、空性四个维度上，都有"掌握度"（懂多少）和"占比"（有多少）两个独立参数
- **差序格局动力学**：人的关系像旋转的星系，层级不同自然会被离心力甩出——不是谁对谁错，是转速不一样了
- **三轴定位系统**：核心驱动层（X轴）× 四性结构（Y轴）× 环境匹配度（Z轴）——构成 36 个常见的人类坐标

详细的原理说明见 `docs/原理说明.md`。
